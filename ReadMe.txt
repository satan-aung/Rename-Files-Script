====================================================
          GEMINI SMART RENAME UTILITY
====================================================

OVERVIEW:
A powerful Windows Batch utility to rename, sort, and 
organize files via the Right-Click context menu.

FEATURES:
- Rename by Sequence (001, 002) or Timestamp.
- Sort by Name, Date Created, Date Modified, or Size.
- Shuffle/Randomize files (great for music/slideshows).
- Extension filtering (e.g., only rename .jpg files).
- Dry Run mode (Preview changes before they happen).
- Undo/Revert functionality via automated log files.

INSTALLATION:
1. Right-click 'Install_Menu.bat' and Run as Administrator.
2. The script will be moved to C:\Scripts\SmartRename.bat.
3. You can now right-click any folder to begin.

REMOVAL:
1. Run 'Uninstall_Menu.bat' as Administrator.
2. This will remove the right-click menu and delete the 
   script from your C:\Scripts folder.

SAFETY:
Always run a 'Dry Run' (Option Y) before committing 
changes to your files.
